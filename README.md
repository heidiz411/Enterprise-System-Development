** Workshop 1 **
ขั้นตอนการทำงานกับ git
1. สร้างโฟลเดอร์ เพื่อใช้ในงาน
2. เปิด termianal (cmd) แล้วไปยัง path ของโฟลเดอร์งานด้วยคำสั่ง
   "cd <ตำแหน่งโฟลเดอร์>"
3. สร้างพื้นที่ git ด้วยคำสั่ง "git init"
4. เพิ่มไฟล์ เริ่มต้นไปที่ git ด้วยคำสั่ง "git add ."
5. เมื่อทำงานเรียบร้อยแล้วต้องเก็บประวัติการทำงาน ใช้คำสั่ง "git commit -m 'ข้อความ'"
6. อัพโหลดขึ้น cloud ด้วยคำสั่ง "git push"
